Is it a roguelike?

Heavy use of procedural generation
Highly tactical
Requires technical skill (dodging, twitch aiming etc.)
Game flow organized around "floors" and/or "rooms"
Game has "runs" which end upon either death or victory
Death is a part of the game flow in a way that is not game over*
Character building with visible stats or abilities
Movement entirely on a grid
ASCII graphics as an option or default

Classic(-style) Roguelike

*I.e. if one must die to access some function of the game, which can affect future runs; this doesn't include The Binding Of Isaac, where unlocks happen in play independent of wins or losses past the first 11 Mom's Heart fights, but it does include Rogue Legacy, where dying is the only way to upgrade your character's various stats.
