quaternions

This module contains the type, which is used to represent quaternions, along with related mathematical operations and functions.

  • quaternion

    Declaration

    pure nothrow @nogc @safe auto quaternion(R)(const R re) if (is(R : double));
    pure nothrow @nogc @safe auto quaternion(R, I, J, K)(const R re, const I i, const J j, const K k) if (is(R : double) && is(I : double) && is(J : double) && is(K : double));

    Helper function that returns a quaternion with the specified parts.

    Parameters

    R

    (template parameter) type of real part of quaternion

    I

    (template parameter) type of i part of quaternion

    J

    (template parameter) type of j part of quaternion

    K

    (template parameter) type of k part of quaternion

    R re

    real part of quaternion to be constructed

    I i

    (optional) i part of quaternion, 0 if omitted.

    J j

    (optional) j part of quaternion, 0 if omitted.

    K k

    (optional) k part of quaternion, 0 if omitted.

    Return Value

    Quaternion instance with all parts set the the values provided as input. If neither re, i, j or k are floating-point numbers, the return type will be Complex!double. Otherwise, the return type is deduced using .

    Examples

    1. auto a = quaternion(1.0); static assert(is(typeof(a) == Quaternion!double)); assert(a.re == 1.0); assert(a.i == 0.0); assert(a.j == 0.0); assert(a.k == 0.0); auto b = quaternion(2.0L); static assert(is(typeof(b) == Quaternion!real)); assert(b.re == 2.0L); assert(b.i == 0.0); assert(b.j == 0.0); assert(b.k == 0.0); auto c = quaternion(1.0, 2.0, 1.5, 3.0); static assert(is(typeof(c) == Quaternion!double)); assert(c.re == 1.0); assert(c.i == 2.0); assert(c.j == 1.5); assert(c.k == 3.0); auto d = quaternion(3.0, 4.0L,2.0,5.0L); static assert(is(typeof(d) == Quaternion!real)); assert(d.re == 3.0); assert(d.i == 4.0); assert(d.j == 2.0); assert(d.k == 5.0); auto e = quaternion(1); static assert(is(typeof(e) == Quaternion!double)); assert(e.re == 1); assert(e.i == 0.0); assert(e.j == 0.0); assert(e.k == 0.0); auto f = quaternion(1L, 2, 3, 4); static assert(is(typeof(f) == Quaternion!double)); assert(f.re == 1L); assert(f.i == 2); assert(f.j == 3); assert(f.k == 4); auto g = quaternion(3, 4.0L, 5, 6); static assert(is(typeof(g) == Quaternion!real)); assert(g.re == 3); assert(g.i == 4.0); assert(g.j == 5); assert(g.k == 6);

  • conjugate

    Declaration

    Quaternion!T conjugate(T)(Quaternion!T p);
    Quaternion!(CommonType!(R, T)) conjugate(R, T)(Quaternion!R p, Quaternion!T q);

    Parameters

    Quaternion!T p

    A quaternion.

    Quaternion!T q

    A unit quaternion.

    Return Value

    p conjugated with q. If only one quaternion is given, it will simply return that quaternion's conjugate.

    Examples

    1. auto q = quaternion(0.5,0.5,0.5,0.5); auto p = quaternion(0,1,1,1).normed; assert(q.conjugate==quaternion(0.5,-0.5,-0.5,-0.5)); enum SQRT1_3 = std.math.sqrt(1.0/3); assert(conjugate(p,q)==quaternion(0,SQRT1_3,SQRT1_3,SQRT1_3));
